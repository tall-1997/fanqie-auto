✅ v6.0.0 深度Hook功能

  • 深度扫描 - 扫描所有签到/任务/奖励相关类
  • 方法Hook - Hook所有相关方法并记录调用
  • 参数记录 - 记录所有方法参数和返回值
  • 定时调用 - 每60秒尝试调用签到方法
  • 状态输出 - 定期输出已找到的方法数量

🔍 扫描的类

GoldCoinRepo - 金币签到 LongSignInKlayViewModel - 长期签到 NewUserSevenDaySignInViewModel - 新用户签到 AlarmClockRepository - 闹钟签到 TaskDoneDataRepository - 任务完成 TreasureTaskDoneDataRepository - 宝箱任务 SecondFloorDivideGoldCoinViewModel - 金币任务

⚠️ 使用说明

1. 下载并安装 APK

2. 在 LSPosed 中启用模块

3. 设置作用域为「番茄免费小说」

4. 重启番茄小说应用

5. 查看日志了解Hook情况

📊 查看日志

adb logcat -s Xposed:* FanqieAuto:* # 日志会显示： # [FanqieAuto] 已找到 X 个相关方法 # [FanqieAuto] [=== 方法调用 ===] # [FanqieAuto] 类: xxx # [FanqieAuto] 方法: xxx # [FanqieAuto] 参数: xxx # [FanqieAuto] 返回: xxx
