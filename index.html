功能特性

  • 自动监控签到接口
  • 自动监控任务接口
  • 智能识别金币奖励
  • HTTP请求日志记录
  • 签到结果实时通知
  • 支持多种签到方式
📥 下载源代码 📚 libxposed API

⚠️ 使用说明

本模块需要 LSPosed 框架支持，请确保已正确安装 LSPosed。

使用前请在 LSPosed 中启用模块并设置作用域为「番茄免费小说」。

技术细节

目标应用 com.dragon.read (番茄小说)
Xposed API libxposed API 101
最低Android版本 Android 8.0 (API 26)
Hook点 OkHttp3 Request Builder
监控字段 status, coin, gold, reward, obtain
